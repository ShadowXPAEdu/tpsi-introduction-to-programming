# Introduction to Programming

This subject focused on learning how to program. The language used in this subject was **C**.

## Assignment

This subject did not have any assignments, so this repository contains only what was done during class.

## Final Grade

The grades go from **0** to **20**. They were divided into **Theory** and **Practical**. The formula for the final grade is **`0.7 * T + 0.3 * P`**, where **T** is the **Theory** grade and **P** is the **Practical** grade.  
My **T** grade was **18.93**, and my **P** grade was **20**.  
So the **final grade** was **19**.

## Conclusions

With this subject I was able to learn how to program. It was a very good introductory class, and I believe **C** is one of, if not the best languages to start learning programming.
